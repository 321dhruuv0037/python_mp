Line Chart

A line graph also known as a line plot or a line chart is a graph that uses lines to connect individual data points.

Why have we used line chart?

1.Shows trends over time: A line chart is ideal for displaying trends over time, as it connects data points with a line, allowing viewers to easily see how the data changes over the given time period.

2.Simplifies data: A line chart can simplify complex data by presenting it in a clear and straightforward manner. By plotting the average percentile values against time, viewers can easily see the overall trend and make informed decisions.

3.Easy to compare: A line chart can effectively compare multiple streams or data series on a single graph. By plotting the average percentile values for each stream on the same chart, it is easy to see how they compare to one another.

4.Highlight outliers: A line chart can help identify outliers or abnormal values, which can be visually represented as points that deviate significantly from the trend line.
